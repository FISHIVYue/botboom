## Asset policy

- Prefer CC0, MIT, BSD, Apache-2.0, or SIL OFL.
- Record every asset in `THIRD_PARTY_NOTICES.md`.
- Do not use a GitHub repository only because it is public; public does not mean freely licensed.
- Do not let Codex scrape random images from search results.
- Do not use hotlinked assets.
- Download assets once, store them locally, and commit only when the hackathon rules permit.
- Use one coherent UI pack instead of mixing several packs.
